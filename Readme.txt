The web application is a simple implementation of Registration Form data insertion,show and search feature using Php and Mysql.
To successfully run the web application one must follow these steps:

1. Download and Install xampp.

2. After Installing, Copy and Paste the whole folder <2017-2-60-160>  to this directory: C:\xampp\htdocs

3. Open Xampp program and then start Apache and Mysql.

4. Click on "Admin" button of "Mysql". Your Browser will open and you will be redirected to phpmyadmin page.

5. Create a new database with exactly the same name as the <testdb.sql> file situated at 
location C:\xampp\htdocs\2017-2-60-160

6. Click on "Import" tab. From the browse option click on choose file and select <testdb.sql> file 
from location C:\xampp\htdocs\2017-2-60-160. 
Finally, Scroll down and click on go. 

7. Now open a browser and in address bar, type: localhost/2017-2-60-160

8. Enter some information for several time to get started and check.

9. I hope you like it!
